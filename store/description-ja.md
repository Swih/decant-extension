# Decant — Chrome Web Store 説明文 (JA)

## 短い説明 (132文字以内)
あらゆるWebページからMarkdownとJSONを抽出。ChatGPT・Claude・Notionに最適なクリーンテキストをワンクリックで。

## 詳細説明

あらゆるWebページからクリーンなコンテンツをワンクリックで抽出し、AIワークフローに活用。

今日のWeb閲覧は、広告、ポップアップ、Cookie バナー、無限のサイドバーとの戦いです。Decantはすべてのノイズを除去します。記事、ブログ投稿、ドキュメントを瞬時にピュアでフォーマットされたテキストに変換 — ChatGPT、Claude、Gemini、その他のLLMにすぐ貼り付けられます。

ChatGPTやNotionにコードブロックや壊れたリンクを手動でコピペするのはもうやめましょう。Decantがすべて自動処理します。

🏆 なぜDECANT？

AI対応フォーマット：Markdown、JSON、またはMCP（Model Context Protocol）で出力 — ChatGPT、Claude、Geminiなどに最適化。不要なHTMLでコンテキスト制限を無駄にしません。
100%ローカル処理：MozillaのReadabilityアルゴリズムをブラウザ内で直接使用。外部サーバーにデータは送信されません。アカウント不要。

⚡ 主な機能

Markdownエクスポート：見出し、コードブロック、リンク、画像付きのクリーンテキスト。ChatGPT、Claudeへの貼り付けやObsidian、Notionへの保存に最適。
JSONエクスポート：メタデータ付き構造化データ — 開発者、自動ワークフロー、APIに最適。
MCP出力（Model Context Protocol）：ClaudeとAIエージェントワークフローのネイティブフォーマット。MCP対応ツールとAIアシスタントにWebコンテンツを直接投入。
スマート抽出：テーブル、日付、メール、価格の自動検出。
超高速：クリック不要。Ctrl+Shift+Cを押すだけで抽出してコピー。

👨‍💻 こんな方に最適

プロンプトエンジニア & ChatGPT/Claudeユーザー：ドキュメントページ全体をクリーンなMarkdownでChatGPT、Claude、Gemini、Copilotに貼り付け。
研究者 & 学生：Obsidian、Roam、Notionにウェブサイトの雑然さなしで記事を保存。
AI開発者 & MCPユーザー：Claude エージェントとAIツールチェーン用にMCPフォーマットでWebコンテンツをエクスポート。
開発者：JSON出力を軽量でプライバシー重視のWebスクレーパーとして使用。

⌨️ キーボードショートカット

Ctrl+Shift+E (Mac: Cmd+Shift+E) — Decantサイドパネルを開く
Ctrl+Shift+C (Mac: Cmd+Shift+C) — ページを抽出して直接コピー
Ctrl+Shift+S (Mac: Cmd+Shift+S) — 抽出してファイルに保存 (.md / .json)

🔒 プライバシー宣言：Decantは'activeTab'権限のみを要求します。トラッキングなし。GitHubでオープンソース（MITライセンス）。
