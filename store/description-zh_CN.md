# Decant — Chrome Web Store 描述 (ZH-CN)

## 简短描述 (132字符以内)
从任意网页提取Markdown和JSON。为ChatGPT、Claude、Notion及AI工作流提供干净文本，一键完成。

## 详细描述

一键从任何网页提取干净内容，驱动您的AI工作流。

如今浏览网页意味着与广告、弹窗、Cookie横幅和无尽的侧边栏作斗争。Decant去除所有噪音。它即时将任何文章、博客帖子或文档转换为纯净、格式化的文本——可直接粘贴到ChatGPT、Claude、Gemini或任何LLM中。

别再手动复制粘贴代码块和损坏的链接到ChatGPT或Notion了。Decant为您完成所有工作。

🏆 为什么选择DECANT？

AI就绪格式：以Markdown、JSON或MCP（Model Context Protocol）输出——完美结构化，适用于ChatGPT、Claude、Gemini等LLM，不会用无用HTML浪费上下文窗口。
100%本地隐私：我们直接在您的浏览器中使用Mozilla的Readability算法。不向任何外部服务器发送数据。无需账户。

⚡ 核心功能

Markdown导出：干净、可读的文本，保留标题、代码块、链接和图片。完美适用于粘贴到ChatGPT、Claude，或保存到Obsidian和Notion。
JSON导出：带元数据的结构化数据——适用于开发者、自动化工作流和API。
MCP输出（Model Context Protocol）：Claude和AI代理工作流的原生格式。直接将网页内容输入MCP兼容工具和AI助手。
智能提取：自动检测表格、日期、邮箱和价格。
极速：零点击操作。只需按Ctrl+Shift+C即可提取并复制。

👨‍💻 适用人群

Prompt工程师和ChatGPT/Claude用户：抓取整个文档页面，以干净的Markdown粘贴到ChatGPT、Claude、Gemini或Copilot中。
研究人员和学生：将文章保存到Obsidian、Roam或Notion，无网站杂乱。
AI开发者和MCP用户：以MCP格式导出网页内容，用于Claude代理和AI工具链。
开发者：将JSON输出用作轻量级、注重隐私的网页爬虫。

⌨️ 键盘快捷键

Ctrl+Shift+E (Mac: Cmd+Shift+E) — 打开Decant侧面板
Ctrl+Shift+C (Mac: Cmd+Shift+C) — 提取页面并直接复制
Ctrl+Shift+S (Mac: Cmd+Shift+S) — 提取并保存为文件 (.md / .json)

🔒 隐私承诺：Decant仅需'activeTab'权限。我们不追踪您。GitHub开源（MIT许可证）。
